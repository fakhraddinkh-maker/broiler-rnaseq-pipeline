## 📌 Project Overview

This project implements a **fully reproducible RNA-seq analysis pipeline** for broiler chicken transcriptomic data.

It performs:
- Differential gene functional interpretation
- GO Biological Process enrichment
- KEGG pathway analysis
- Publication-quality visualization

This workflow is designed for **PhD-level research applications and scientific publication**.

---

## 🧠 Biological Question

What biological processes and pathways are significantly affected in differentially expressed genes (DEGs) in broiler chickens?
